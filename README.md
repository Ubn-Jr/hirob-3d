# Hirob Project 3D and 2D Drawings Repository

This repository is dedicated to the 3D and 2D aspects of the Hirob Project.

## About the Project
This GitHub repository contains the 3D and 2D drawings related to the Hirob Project's product. Our objective is to fulfill the 3D and 2D design requirements necessary for the project.

## Our Tasks
Within this repository, we are focusing on the following tasks:

- **3D Modeling:** Creating and updating 3D models of the product.
- **2D Designs:** Preparing 2D drawings and graphics for the product.
- **Documentation:** Providing detailed documentation for each drawing or modeling process.
- **Progress Tracking:** Monitoring the project's progress and delivering regular updates.

We will utilize this repository to share information and tasks related to our project.
